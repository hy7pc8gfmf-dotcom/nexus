# Nexus 模块化架构

## 三系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                     coordinator.exe                         │
│              生命周期管理 / 统一启动入口                       │
└────┬──────────┬──────────┬─────────────────────────────────┘
     │          │          │
┌────▼────┐ ┌──▼───┐ ┌───▼──────┐
│ 神经    │ │ 逻辑  │ │ 量子     │
│ 系统    │ │ 系统  │ │ 系统     │
├─────────┤ ├──────┤ ├──────────┤
│env_check │ │logic │ │quantum   │
│daemon   │ │      │ │          │
│core     │ │      │ │          │
│algo     │ │      │ │          │
│psyche   │ │      │ │          │
│bridge   │ │      │ │          │
└─────────┘ └──────┘ └──────────┘

  7 EXE       1 EXE       1 EXE
```

## 安装约定

### 目录结构（推荐）

```
D:/nexus/                    # 安装目录 (可任意)
├── bin/                     # EXE 文件
│   ├── coordinator.exe      # 生命周期管理器
│   ├── env_checker.exe      # 环境检测
│   ├── daemon.exe           # 后台守护
│   ├── core.exe             # 推理引擎
│   ├── algo.exe             # 算法池
│   ├── psyche.exe           # 意识层
│   ├── bridge.exe           # 桥接层
│   ├── quantum.exe          # 量子系统
│   ├── logic.exe            # 逻辑系统
│   └── algo_engine.exe      # 隔离子进程
└── data/                    # 运行时数据 (自动创建)
    ├── env.json
    ├── daemon.json
    ├── core_state.json
    └── ...
```

### 数据目录解析

EXE 查找数据目录的优先级：

```
1. 命令行: --data-dir <path>
2. 环境变量: %NEXUS_DATA_DIR%
3. 默认: {exe_dir}/../data/
```

## 模块化设计原则

```
┌─────────────────────────────────────────────────────────┐
│  原则 1: 各系统独立编译、独立运行、独立发布                │
│                                                         │
│  原则 2: 统一通信协议 (文件 IPC + mmap)                  │
│         所有状态文件格式遵循 nexus-state-v1 schema       │
│                                                         │
│  原则 3: 数据目录可配置                                  │
│         通过 --data-dir 或 NEXUS_DATA_DIR 变量指定       │
│                                                         │
│  原则 4: coordinator 只协调不依赖                         │
│         各 EXE 可脱离 coordinator 独立启动                │
└─────────────────────────────────────────────────────────┘
```

## 各系统边界

| 系统 | EXE | 职责 | 依赖 |
|:--|:--|:--|:--|
| 神经 | 7 EXE | 推理/算法/意识/桥接 | 无 |
| 逻辑 | logic.exe | 种子空间/Coq证明/逻辑推理 | env.json |
| 量子 | quantum.exe | 21子系统/涌现/信念 | env.json |
